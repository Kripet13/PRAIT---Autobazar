#Autobazar

Jednoduchá webová aplikace vytvořená v HTML, CSS a JavaScriptu.

Aplikace simuluje jednoduchý autobazar, ve kterém lze přidávat auta do seznamu, zobrazovat je na stránce, mazat je a počítat jejich průměrnou cenu.

##Funkce aplikace

přidání auta pomocí formuláře

zobrazení seznamu aut

možnost smazat auto ze seznamu

zobrazení počtu aut

výpočet průměrné ceny

kontrola zadaných údajů (rok výroby, cena, povinná pole)

možnost zadání slevy 10 % pomocí checkboxu

U zlevněných aut se zobrazuje původní cena i cena po slevě.
Průměrná cena se počítá z konečných cen (po započtení slevy).

##Použité technologie

HTML

CSS

JavaScript

objektově orientované programování (třídy Auto a Autobazar)

##Spuštění aplikace

Stáhněte nebo naklonujte repozitář.

Otevřete soubor index.html v internetovém prohlížeči.

Aplikace funguje bez serveru a bez externích knihoven.

##Struktura projektu

index.html – hlavní stránka

style.css – vzhled aplikace

js/main.js – hlavní logika aplikace

js/auto.js – třída Auto

js/autobazar.js – třída Autobazar
